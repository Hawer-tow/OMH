	You are software engineering and apple company has tasked you to design some requirement specifications 
that they can implement in order to ensure swift transition of user's other companies. As a software engineer

THE FUNCTIONAL REQUIREMENTS 
1) The users should be able to access other companies/machine operating systems with ease. 
2) The additional functionalities to the system should take less memory space. 
3) The system should be able to connect with other companies quickly. 
4) The system should ensure smooth transition of the system from the apple machine structure to other 
structures. 
5) The system should be able to provide the user with easy methodology in the operation of the system 
once it is shifted from the apples system to the other industry systems. 
6) The system should provide easy maintenance structures. 
7) The system should pair with new required hardware devices. 
8) The apples system relational modules should be built in modules depending on the factorial relational 
models. 
9) The apples system should under-go some testing to ensure the systems modules that have been built 
differently are able to integrate. 
10) The apples system should be able to provide an easy-to-use structure for ease of navigation by the 
user. 
11) Apples system should be able to able to maintain the data integrity in case of the system failure of shut 
down. 
12) The system should ensure that un-authorized user data is kept safe from the other companies unless 
under the users consent or through proper authorization.
